# react-webpack-boilerplate

React + Webpack + Typescript + Styled Components

# How to start

- Clone repository: `git clone https://github.com/tatinacher/react-webpack-boilerplate.git`
- Run `yarn install` or `npm install` to install dependencies
- To run in dev mode `yarn start` or `npm start`
- To build project `yarn build` or `npm build`
